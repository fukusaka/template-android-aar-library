template-android-aar-library
------------------------------

* Android Studio にて empty project を指定して新規にプロジェクト作成
* 追加モジュールとして Android Library を新規にプロジェクトに追加
* buildSrc に共用処理を追加
* GitHub Packages にアップロードワークフロー追加
* sample code を追加
